Fifthtry
Using Spellcraft on Fifthtry
Start from template
You can start from the spellcraft template inside fifthtry and your site will be built and managed with fifthtry.
Editing Pages
You can edit pages using the fifthtry editor. You can also use the fifthtry editor to create new pages.
Configuring Defaults
You can edit the default settings for your website in the fifthtry settings of the particular website.
